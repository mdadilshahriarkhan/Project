===============(Client Server)==========Server01:192.168.241.99 & 192.168.241.31-(Node server)=====================================


Step 1: Install Node Exporter on Each MySQL Server
Node Exporter collects hardware and OS metrics from Linux systems.

On 192.168.241.99 (mysql-RnD) and 192.168.241.31 (mysql-DB-reporting)
1.1 Download and Install Node Exporter:

cd /opt
sudo wget https://github.com/prometheus/node_exporter/releases/download/v1.8.2/node_exporter-1.8.2.linux-amd64.tar.gz
sudo tar -xvf node_exporter-1.8.2.linux-amd64.tar.gz
sudo mv node_exporter-1.8.2.linux-amd64 node_exporter
sudo useradd --no-create-home --shell /bin/false node_exporter

1.2 Create systemd service:
sudo cat > /etc/systemd/system/node_exporter.service << EOF
[Unit]
Description=Prometheus Node Exporter
After=network.target

[Service]
User=node_exporter
Group=node_exporter
Type=simple
ExecStart=/opt/node_exporter/node_exporter --web.listen-address=:9100

[Install]
WantedBy=multi-user.target
EOF

1.3 Start and enable Node Exporter:
sudo systemctl daemon-reload
sudo systemctl start node_exporter
sudo systemctl enable node_exporter
sudo systemctl status node_exporter

1.4 Configure firewall (if enabled):
sudo firewall-cmd --zone=public --add-port=9100/tcp --permanent
sudo firewall-cmd --reload

========(Monitoring Server)=======Server-02:192.168.241.38 (Prometheus&Grafana)====================================

Step 3: Update Prometheus Configuration
Now add the Node Exporter targets to your Prometheus configuration on 192.169.241.38.

3.1 Edit Prometheus config:
sudo vi /etc/prometheus/prometheus.yml

3.2 Add a new job for Node Exporters:
scrape_configs:
  # MySQL job (already exists)
  - job_name: 'mysql'
    static_configs:
      - targets: ['192.168.241.99:9104']
        labels:
          instance: 'mysql-RnD_241.99'
      - targets: ['192.168.241.31:9104']
        labels:
          instance: 'mysql-DB-reporting_241.31'

  # NEW: Node Exporter job for OS metrics
  - job_name: 'node'
    static_configs:
      - targets: ['192.168.241.99:9100']
        labels:
          instance: 'mysql-RnD_241.99'
      - targets: ['192.168.241.31:9100']
        labels:
          instance: 'mysql-DB-reporting_241.31'

3.3 restart Prometheus:
systemctl restart prometheus

3.4 Verify targets in Prometheus:
http://192.169.241.38:9090/targets

You should now see:

Both MySQL targets (port 9104) - UP

Both Node targets (port 9100) - UP

Step 5: Essential PromQL Queries for Custom Panels
If you want to build custom panels, here are the key queries for each metric:

CPU Monitoring
# CPU utilization (as percentage)
100 - (avg by(instance) (rate(node_cpu_seconds_total{mode="idle"}[5m])) * 100)

# System load average (1 min)
node_load1

# CPU by mode (user, system, etc.)
rate(node_cpu_seconds_total[5m])

RAM Monitoring
# Available memory (bytes)
node_memory_MemAvailable_bytes

# Memory utilization percentage
((node_memory_MemTotal_bytes - node_memory_MemAvailable_bytes) / node_memory_MemTotal_bytes) * 100

# Swap usage
node_memory_SwapFree_bytes / node_memory_SwapTotal_bytes * 100

Disk / Hard Disk Monitoring
# Disk space utilization
((node_filesystem_size_bytes{mountpoint="/"} - node_filesystem_free_bytes{mountpoint="/"}) / node_filesystem_size_bytes{mountpoint="/"}) * 100

# Disk read/write rate (MB/s)
rate(node_disk_read_bytes_total[5m]) / 1024 / 1024
rate(node_disk_written_bytes_total[5m]) / 1024 / 1024

# Disk I/O time
rate(node_disk_io_time_seconds_total[5m])

Slow Query Monitoring
# Slow queries per second
rate(mysql_global_status_slow_queries[5m])

# Slow queries over time (cumulative)
increase(mysql_global_status_slow_queries[1h])

# QPS (Queries Per Second)
rate(mysql_global_status_queries[5m])

Connection Monitoring
# Current active connections
mysql_global_status_threads_connected

# Connection utilization (percentage of max)
(mysql_global_status_threads_connected / mysql_global_variables_max_connections) * 100

# Aborted connections (potential issues)
rate(mysql_global_status_aborted_connects[5m])

Buffer Pool (InnoDB Performance)
# InnoDB buffer pool usage percentage
(mysql_global_status_innodb_buffer_pool_pages_data / (mysql_global_status_innodb_buffer_pool_pages_data + mysql_global_status_innodb_buffer_pool_pages_free)) * 100
